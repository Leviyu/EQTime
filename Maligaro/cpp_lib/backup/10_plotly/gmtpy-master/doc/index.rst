
GmtPy - A Python interface to GMT
=================================

GmtPy provides seamless integration of `GMT <http://gmt.soest.hawaii.edu/>`_ plotting into Python programs. On top of that it provides (in an opt-in fashion): autoscaling, automatic tick increment determination, layout management, and more.
